ScriptTrigger

This project is intended to automate developer’s testing experience by automation.

Feel free to contribute.

State of work

This project is still “work in progress”.

Command line

      scriptTrigger [-h | -t=TN -v=V [-s=SN|-c=CMD] -l=LIS]

      -h            : help (also /? and --help)

      -t=TRIGGERNAME: sets the trigger-value (see below)
      -v=VALUE      : sets the  value (string)
      
      -e=EXECTYPE   : execution type
      -a=ACTION     : actions to execute with EXECTYPE
      -s=SCRIPTNAME : sets the script name (use absolute paths); short for -e=SCRIPT -a=SCRIPTNAME
      -c=COMMANDS   : sets the commands to execute; short for -e=COMMAND -a=COMMANDS

      -l=True|False : sets whether to listen on startup or not

      TriggerNames: 
        OpenPort    : checks if the application can connect to a port using tcp
